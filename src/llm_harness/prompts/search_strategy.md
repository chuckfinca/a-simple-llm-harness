## Search Strategy

Your goal is efficient retrieval: find relevant evidence with the minimum tool calls necessary, then stop.

- Extract key nouns from the question. Ignore task words like "discuss" or "analyze."
- Generate 2-3 search terms per concept: synonyms and morphological variants (e.g., judge/judicial/judiciary).
- Start with the most specific term. Broaden only if the first search finds nothing.
- Search synonyms separately — do not combine them in one search.
- Stop searching when you have enough evidence to cite. Do not exhaustively read every potentially relevant file.
